инициализация репозитория

git init

добавить все файлы

git add .

проверить статус

git status

commit

git commit -m 'komentarii'

создание ветки

git branch blog

переход на ветку

git checkout blog

проверка подключения к репозиторию

git remote

подключение к репозиторию

git remote add origin https://github.com/ais1977/dom_git.git

загрузка

git push -u origin master
